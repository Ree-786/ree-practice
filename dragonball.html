Learn ninja coding and good cooding

Ninja code

- Make code as short as possible. - Use one-letter variables. - Use abbreviations everywhere. - Name variables by type instead of meaning. - Use similar variable names like 'data' and 'date'. - Reuse the same variable names. - Add unnecessary underscores. - Avoid comments completely. - Mix tabs and spaces. - Write giant powerful functions that do many things at once.

Good Coding

- Use curly brackets appropriately with spacing before and after. - Keep horizontal lines short; break long code lines to avoid scrolling. - Use consistent indentation. - Use semicolons and do not skip them. - Avoid nesting too many levels (like deep loops). - Declare functions properly. - Follow a styling guide for your code. - Use automatic linters to format code. - Write comments that make sense. - Give variables appropriate and meaningful names.
